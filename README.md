## Installation

```bash
$ npm install
```

## Running the app

```bash
# development mode
$ npm run start:dev

# staging mode
$ npm run start:stage

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Linting and formatting

```bash
# format code
$ npm run format

# lint and fix
$ npm run lint

# format code and lint
$ npm run lint:fix
```
